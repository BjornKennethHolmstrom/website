## Project Overview

The UST (**U**niversal **S**earch **T**ree) project implements a flexible and powerful search tree structure for large-scale data organization and retrieval. It is a versatile tool that can represent and navigate complex hierarchical relationships across various domains, from scientific evolution to linguistic development. The project now includes advanced implementations that combine UST with neural networks, pushing the boundaries of adaptive and hierarchical search structures.
