## Development Overview

CommuniTree is a community engagement platform aimed at fostering collaboration between municipalities and residents through volunteer tasks, paid projects, intergenerational collaboration and community events.

### Current Status

- Initial project setup completed

- Core features implemented:
    - User, Project, Event, and Community Management Systems (CRUD operations)
    
    - Community event calendar with RSVPs
    
    - Role-based access control
    
    - Messaging, notifications, and profiles with customization options
    
    - Weather and location-based theming
    
    - Multi-language support (English and Swedish)
    
    - Search and filter functionality for projects
    
    - Pagination for project listing
    
    - User authentication with JWT
    
    - Protected routes for authenticated users

### Tech Stack

React, Express.js, PostgreSQL, Chakra UI, OpenWeatherMap API.
