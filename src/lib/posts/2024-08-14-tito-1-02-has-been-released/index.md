TiTo is a time tracker tool for managing time spent on various projects.

Features:

- Interface for creating projects, tracking time with a start/stop timer, and manually adding time entries.

- Sorting projects by drag and drop

- Sorting time entries by newest or oldest first

- Export and import all data for backups or transfers

- A visual chart of time spent across projects.

TiTo runs directly in the browser and stores all data locally.  

Upcoming features

- Button to clear all time entries for a project

- View summarizing time spent per week & month

Download TiTo here: [https://bjornkennethholmstrom.wordpress.com/tito/](https://bjornkennethholmstrom.wordpress.com/tito/)

follow the installation instructions
