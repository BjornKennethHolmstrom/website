TiTo is a time tracker tool for managing time spent on various projects.

TiTo runs directly in the browser and stores all data locally.

Download TiTo here: [https://bjornkennethholmstrom.wordpress.com/tito/](https://bjornkennethholmstrom.wordpress.com/tito/)

and follow the installation instructions.

Features added in latest update:

- Support for screen reader and keyboard navigation (tested with Orca in Chrome on Linux Mint)

- Multilingual support (English, Swedish, Spanish and Basque)

More features:

- A clean three-column layout

- Create and manage multiple projects

- Start, pause, and stop a timer for active time tracking

- Add manual time entries

- Edit project names

- Reorder projects with drag and drop

- View and edit time entries

- Clear all entries-button for time list entries

- Paged view of time entries

- Chart tabs displaying time spent on projects, total time and ranges

- Monthly and weekly report features with export to CSV, PDF and markup

- Local data storage using IndexedDB

- Import/Export of database from/into .JSON-file

- Offline functionality

Upcoming features

- Improved aesthetics

- Dark mode

- Customizable report templates

- Time goals

- More data visualization
